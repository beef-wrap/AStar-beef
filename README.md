# About

A Beef wrapper for [AStar](https://github.com/BigZaphod/AStar), a pathfinding algorithm implemented in pure C.

# Building static libs

```sh
zig build --prefix libs
```

# Running the example

Clone the repo, and open up the workspace in the Beef IDE, and run.
